# Предсказание калорийности блюд

Мультимодальная модель (EfficientNet + DistilBERT + масса порции) для регрессии калорий.

## Структура
- `scripts/` – весь код на Python
- `notebook.ipynb` – исследовательский анализ и демонстрация обучения
- `data/` – папка с данными (не включена в репозиторий)

## Установка зависимостей
```
pip install -r requirements.txt
```

Данные
Скачайте датасет Nutrition5k и поместите в data/ так, чтобы там были файлы dish.csv, ingredients.csv и папка images/.

Запуск обучения
Откройте notebook.ipynb и последовательно выполните ячейки.
Либо запустите скрипт:

```
python scripts/train.py
```

Результат
Достигнутая MAE на тесте: < 50 (уточните своё значение).
